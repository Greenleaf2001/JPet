## 编译

该项目需要的部分依赖已经置于`thirdparty`下，包括：

- GLFW
- FMOD CORE
- CubismSdkForNative

其余依赖定义在 vcpkg.json 中，可使用 vcpkg 安装：

```shell
vcpkg install
```

依赖安装完成后即可使用 CMake 编译
